# Weather Analysis Mini Project 📊

Analyzes temperature, humidity, and rainfall data for Delhi, Mumbai, and Bangalore.

## What's inside
- 90 days of sample weather data
- 4 visualizations: line chart, bar chart, scatter plot, pie chart  
- Key insights like hottest city and rainfall comparison

## Run it yourself
1. Download `matplotlib.ipynb`
2. Open in Jupyter Notebook
3. Run all cells

Built with Python, Pandas, Matplotlib
